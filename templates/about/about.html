Data

ESPN
 From the ESPN site, we obtained general statistics for each player. We started with teams and scraped every player from each team roster using BeautifulSoup.
xmlstats
 We just used the already provided nba API. We pulled their data as a JSON, which allows me to programatically access the data.
Twitter
 We went to the twitter search bar and inputted the name. We checked whether after we searched that name, if there are any official twitter accounts grabbed from the search query. If there are, what is typically the case is the first official twitter we got is usually the twitter of the person we want using BeautifulSoup.
YouTube
 For YouTube, we searched the players name and grabbed videos from the resulting page using BeautifulSoup.
